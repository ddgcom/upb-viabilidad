Fuentes de Financiación Disponibles

Cálculo de Inversión (CAPEX)

Este valor se obtiene del módulo de cálculo de potencial energético.

Referencia Solar: 800 - 1,200 USD/kW

Referencia Eólico: 1,200 - 1,800 USD/kW

Referencia Hidro: 1,500 - 3,000 USD/kW

Referencia Biomasa: 2,000 - 4,000 USD/kW

CAPEX = Capacidad (kW) x Costo de instalación (USD/kW)

Proyección de Ingresos y Gastos Operativos

Típicamente entre 1% y 3% del CAPEX anual para sistemas solares.

Indicadores de Éxito Financiero

Los siguientes indicadores se calcularán automáticamente:

  • VPN (Valor Presente Neto): Debe ser mayor a 0 para ser viable
  • Payback: Período de recuperación de la inversión
  • TIR: Tasa Interna de Retorno

Típicamente 25 años para sistemas solares, 20-30 para hidroeléctricos.

Para proyectos de energía renovable, típicamente entre 8% y 12%.

Interpretación de Resultados

Alta (≥ 4.0): La comunidad cuenta con condiciones sólidas para avanzar a la fase de diseño integral.
Media (3.0 - 3.99): El proyecto es potencialmente viable, pero requiere ajustes previos en algunos componentes.
Baja (< 3.0): La comunidad presenta barreras críticas. Se sugiere replantear el diseño o implementar acciones de mejora.
